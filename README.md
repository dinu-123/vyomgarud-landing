VyomGarud – Landing Page

This project is a modern and responsive landing page created for the VyomGarud Web Developer Intern Assessment.
The design reflects precision engineering, advanced drone technology, and a military-grade, futuristic aesthetic.

Tech Stack

React + Vite

Tailwind CSS

Framer Motion

Poppins / Inter fonts


Sections Included
1. Hero

Bold “VyomGarud” title

Tagline + CTA button

Clean, dark, professional design

2. About

Mission of VyomGarud

Short and impactful content

3. Capabilities / Products

Three capability cards

Modern hover animations

Responsive layout

4. Highlights

Three key features

Subtle fade-up animations

5. Footer

Simple, professional layout

Contact info placeholder


Responsiveness

Fully responsive for mobile, tablet, and desktop

Fluid layouts using Tailwind utilities
